# NoMoreHax
AntiCheat server-side para Minecraft 1.8.9 (Spigot/Paper, ProtocolLib), 100% por análise de pacotes. 17 Checks de Combate, Movimento, Mundo e Pacote - com Speed/Fly preditivos, Timer e Post via framework de tick-por-transaction, compensação de latência, setback e cancelamento de pacotes ilegais.
